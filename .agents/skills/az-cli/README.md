# az-cli Skill

Scaffold production-ready TypeScript CLI tools with Bun.

## Installation

Copy this directory to your Claude Code skills folder or install via skill package manager.

## Usage

```
/az-cli my-cli "A CLI tool for XYZ"
```

## Features

- **TypeScript** with strict mode
- **Bun** as package manager and build tool
- **citty** for CLI framework (auto --version, --help)
- **consola** for colored logging
- **figlet** for ASCII banners
- **ESM + CJS + Types** output
- **Changesets** for versioning

## Template Structure

See `assets/template/` for the full scaffold template.
